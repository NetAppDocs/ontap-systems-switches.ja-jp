NVIDIA SN2100 スイッチのインストールとメンテナンスについては、キャビネットとレール キットのコンポーネントと部品番号のリストを必ず確認してください。



== キャビネットの詳細

NVIDIA SN2100 スイッチ (X190006/X190106) は、スイッチに付属の標準ブラケットを使用してNVIDIAデュアル/シングル スイッチ キャビネットに取り付けます。
