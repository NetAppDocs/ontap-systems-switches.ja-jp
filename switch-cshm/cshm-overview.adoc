= スイッチヘルスモニターの概要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
イーサネットスイッチヘルスモニター（CSHM）は、ネットワークスイッチの動作状態を正常に保ち、デバッグ目的でスイッチログを収集する役割を担います。
